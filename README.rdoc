= smackr

Smackr is a JRuby gem which wraps the Smack XMPP Java library.

== Installing JRuby 1.6.7 Correctly

  rvm get latest
  rvm reload
  rvm install jruby-1.6.7-ruby19 --1.9
  rvm jruby-1.6.7-ruby19 exec gem install bundler
